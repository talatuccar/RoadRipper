Lite version of the asset "Brick Textures (2K)".This pack contains collection of realistic brick textures.Suitable for buildings, walls, basements, fences, .etc+

Technical Details:-
		- Material: Unity standard material
		- Material count : 4+
		- Texture Size : 256 x 256 (PNG)
		- Texture type : PBR (Albedo/Base, Metallic Roughness, Normal)
		- Tileable : YES
		- Seamless : YES
		

Features:-
	- Many variations of Brick materials included
	- Floor prefabs included (Drag and drop to use)
	- Optmized textures
	- Handpainted reallistic Textures/Materials
	- Seamless and Tileable textures
	- You can use this pack commercially

Folder details:-

	Materials : Where the materials for the model located.
	
	Textures : Contains Textures, that are in (.PNG) file format
	
	(Other folders - Prefabs, Demo)

Notes:
Support URP, HDRP, LWRP, Built-in Render Pipeline.
This project is submitted using Built-in Render Pipeline. If you are not using Built-in Render Pipeline, you need to manually reset the shader of the model to avoid it being pink

How to use:-
	
	Simply drag and drop the prefab to your scene and use it.
	(Prefab(s) located under 
	Asset > Brick Textures LITE> Prefabs).
